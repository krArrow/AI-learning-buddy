

## ğŸš€ Usage

### Quick Start Guide

1. **Create Your First Goal**
   - Click "Create Goal" in sidebar
   - Enter what you want to learn
   - Choose your level (Beginner/Intermediate/Advanced)
   - Set daily time commitment
   - Answer clarification questions

2. **Review Your Roadmap**
   - Navigate to "View Plan"
   - Explore generated modules
   - Review timeline and milestones
   - Check attached resources

3. **Start Learning**
   - Go to "Daily Tasks"
   - Read current task
   - Use provided resources
   - Mark task complete when done

4. **Track Progress**
   - Visit "Progress" page
   - View completion rate
   - Analyze performance metrics
   - Check consistency score

5. **Get Insights**
   - Navigate to "Insights" (after 3+ completed tasks)
   - Read AI-generated analysis
   - Follow recommendations
   - Adapt your approach
